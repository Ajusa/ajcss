Getting started

Getting started with aj.css is pretty easy. You can download the current version of the framework here. After that, simply include it in the head of your html file, and you are good to go! Documentation is on the sidebar. I reccomend reading it in order, as the examples build on each other.

<link rel="stylesheet" type="text/css" href="path/to/aj.min.css">
