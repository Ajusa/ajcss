Getting started

Getting started with aj.css is pretty easy. You can download the current version of the framework here. After that, simply include it in the head of your html file, and you are good to go! Documentation is on the sidebar.

Note that on mobile, viewing the source code is somewhat broken. This is simply due to the nature of the xmp tag, and there isn't much I can do about it. All of the examples still work fine on mobile.
